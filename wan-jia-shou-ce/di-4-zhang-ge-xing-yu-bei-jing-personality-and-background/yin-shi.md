# 隐士

&#x20;你过着隐居生活——栖身于修道院这种与世无争之地，或是孤身一人独自生活。在远离尘世纷扰的生活中，你获得了内心的平静与安宁，或许也从中领悟了苦苦追寻的人生真谛。

**技能熟练项：**医药，宗教

**工具熟练项：**草药工具

**语言：**自选一门语言

**装备：**一个卷轴匣，用来装纳研究成果和祈祷词的笔记，一张冬用毯子，一套普通服装，一套草药工具，以及5 gp

### **隐居生活Life of Seclusion**

&#x20;为何你遗世独立，而又为何重踏尘世？你可以和DM共同商讨决定隐居的原因，也可以依下表选择或掷骰随机决定。

| **d8** | **隐居生活**                   |
| ------ | -------------------------- |
| 1      | 我正在寻求灵魂的启迪。                |
| 2      | 我遵守修会教规，参与集体清修。            |
| 3      | 我因不白之冤而被流放。                |
| 4      | 某件事情改变我人生的大事使我弃世而去。        |
| 5      | 我为了能潜心于艺术、文字或音乐而去寻找一处僻静之地。 |
| 6      | 我要与自然沟通而远离文明世界。            |
| 7      | 我是一座古代遗迹或一件古代遗物的看守人。       |
| 8      | 我是一名追寻圣人，圣地或圣遗物的朝圣者。       |

### **特性：研究发现** **Feature: Discovery**

&#x20;你寂寥的长期隐居令你发现了某些独特而强大的秘密。这项发现的具体内容与你的隐居的理由密切相关。你或许是找到了与宇宙真相，神祇本质，异界大能或是自然之力等相关的伟大真理。你也可能是发现了某处任何人都未曾发现的地点。你可能揭露了久被遗忘的真相，或是发掘出了久远史前的遗物。或者，你也可能是发现了那些一手导致你流放的家伙们相关的情报，而那也就是你重返凡世的契机。

### **建议特征Suggested Characteristics**

&#x20;某些隐士很好的适应了隐居生活，也有些人苦于这种生活的寂寥苦苦等待着同伴到来。无论他们是选择拥抱空寂还是渴望着逃离这一切，独居生活都塑造了他们的态度和理想。少数人甚至会因脱离社会过久而变得疯疯癫癫。

### **其他隐士Other Hermits**

&#x20;这里所述的隐士背景特别之处是其设定了一个避世研习或祈祷的场所。如果你想要扮演一个在荒野离群索居的粗犷隐居者，也可以参考化外之民背景。而如果你想扮演的角色要更偏宗教色彩，则可以参考侍僧背景。或者你也可以扮演一名骗子，假装自己是一个睿智而神圣的人以骗取虔诚蠢蛋们为你提供帮助。\
